TCP Chat Room

This project is one of my favorites! It is a chat room made with python. In this project, I used python to create a server that is constantly listening for clients to connect using the Transmission Control Protocol (TCP). Once a client connects, the server as for a username in order to identify each connected client. When a client sends a message, it first goes to the server and then gets sent to all the connected clients including the sender. As you can see in the picture at the top how the project works.
